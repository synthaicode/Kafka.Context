# Kafka.Context.Application

Application-level configuration and options used by `Kafka.Context`.

Most users should install `Kafka.Context` and let it pull this package transitively.

## What is inside
- `KafkaContextOptions`: programmatic configuration for `KafkaContext`.
- `KsqlDslOptions`: configuration model bound from `appsettings.json` (`KsqlDsl` section).
